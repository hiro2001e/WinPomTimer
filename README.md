# Win Pom Timer

Windows 向けのポモドーロタイマーです。WPF (.NET 8) で実装しています。

## ダウンロード（利用者向け）

配布版は GitHub Releases から取得してください。

- 最新版: <https://github.com/hiro2001e/WinPomTimer/releases/latest>
- ファイル名: `WinPomTimer-win-x64.zip`

展開後、`WinPomTimer.exe` を実行します。

## リポジトリ構成

- `src/`: アプリ本体コード
- `docs/`: 要件・設計・セットアップ手順・利用ガイド
- `assets/icon/`: アプリのアイコン
- `assets/sounds/`: 通知音源
- `assets/images/`: 画像素材
- `downloads/`: 配布 ZIP の作成先（詳細は `downloads/README.md`）

## ドキュメント
