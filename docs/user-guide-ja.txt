Win Pom Timer 使い方ガイド（配布版）

========================================
1. 動作環境
========================================
- Windows 10 / 11 (x64)
- 管理者権限は不要
- .NET の事前インストールは不要（配布版は self-contained）

========================================
2. 起動方法
========================================
1) 配布されたフォルダを任意の場所へ展開する
2) WinPomTimer.exe をダブルクリックする

注意:
- 初回起動時に SmartScreen 警告が出る場合があります（未署名のため）。
- その場合は「詳細情報」→「実行」を選択してください。

========================================
3. 主な機能（詳細）
========================================
- ポモドーロタイマー
  - Work / Short Break / Long Break を管理
  - 開始・一時停止・再開・次へ・リセット
  - 自動遷移（休憩自動開始 / 作業自動開始）設定
- 常駐動作
  - 右上「隠す」や閉じる操作でタスクトレイへ常駐
  - 作業状態を保持したまま再表示可能
- 通知 / 音
  - セッション切替音、作業開始音、休憩前通知音
  - 秒針音（ON/OFF、音量、音源ファイル）
  - Windows 通知 ON/OFF
- 見た目カスタマイズ
  - 背景色、メーター色、テキスト色、フォント
  - 背景透過や不透明度
- タグ / メモ
  - Work 中と Work 終了後（休憩開始前）にタグ選択・メモ入力
  - タグ軸（作業内容 / クライアント / その他）で分類
  - 複数タグ選択時は統計で案分集計
- 統計ダッシュボード
  - 日 / 週 / 月 / 年の集計表示
  - 期間移動（< / > / 今日）
  - 積み上げグラフ、タグ別サマリー総計
  - JSON / CSV / ICS エクスポート

========================================
4. 基本操作
========================================
- 「開始 / 一時停止 / 再開」でタイマー操作
- 「次へ」でセッションをスキップ
- 「リセット」でタイマー状態を初期化
- 右上「隠す」でタスクトレイ常駐
- 設定は右上「設定」から別ウィンドウで変更

========================================
5. データ保存場所
========================================
設定・状態・ログは次のフォルダに保存されます。

- %AppData%\WinPomTimer\

主なファイル:
- %AppData%\WinPomTimer\settings.json   （設定）
- %AppData%\WinPomTimer\state.json      （タイマー状態）
- %AppData%\WinPomTimer\sessions.jsonl  （作業ログ）

※ %AppData% の実体例:
  C:\Users\<ユーザー名>\AppData\Roaming

========================================
6. 利用をやめる場合（削除方法）
========================================
1) アプリを終了する
   - タスクトレイアイコンを右クリックし、終了を選択
2) 配布フォルダを削除する
   - WinPomTimer.exe が入っている配布フォルダ一式を削除
3) ユーザーデータを削除する（完全削除したい場合）
   - %AppData%\WinPomTimer\ フォルダを削除

補足:
- ログを残したい場合は、削除前に統計画面からエクスポートしてください。
- 本アプリはスタートアップ自動登録を行いません。

========================================
7. よくある質問
========================================
Q. 「×」で閉じると終了しますか？
A. いいえ。トレイへ格納されます。完全終了はトレイメニューから行ってください。

Q. ダブルクリックしても画面が見えません。
A. 既にトレイ常駐している可能性があります。タスクトレイを確認してください。
