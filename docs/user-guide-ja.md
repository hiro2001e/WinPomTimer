# Win Pom Timer 使い方ガイド（配布版）

## 1. 動作環境

- Windows 10 / 11 (x64)
- 管理者権限は不要
- .NET の事前インストールは不要（配布版は self-contained）

## 2. 起動方法

1. 配布されたフォルダを任意の場所へ展開する
2. `WinPomTimer.exe` をダブルクリックする

注意:

- 初回起動時に SmartScreen 警告が出る場合がある（未署名のため）
- その場合は `詳細情報` -> `実行` を選択する

## 3. 基本操作

- `開始 / 一時停止 / 再開` でタイマー操作
- `次へ` でセッションをスキップ
- `リセット` でタイマー状態を初期化
- 右上 `隠す` でタスクトレイ常駐

## 4. タグとメモ

- タグは Work 中、および Work 終了後（休憩開始前）に選択できる
- セッションメモも同タイミングで編集できる
- 休憩から Work に戻るとタグ選択は自動でクリアされる

## 5. 統計機能

- メイン画面下部のグラフボタンから統計画面を開く
- グラフ種別: `日 / 週 / 月 / 年`
- 期間移動: `<` `>` `今日`
- 集計軸: `作業内容 / クライアント / クライアント×作業`
- グラフは積み上げ表示、右側サマリーで総計を確認可能

## 6. 設定

- 右上 `設定` から各種設定を変更できる
- 主な設定:
  - タイマー時間
  - 音量、秒針音
  - Windows 通知 ON/OFF
  - 見た目（色、フォント、透過）
  - タグ管理（軸設定を含む）

## 7. データ保存場所

以下に設定とログが保存される。

- `%AppData%\WinPomTimer\settings.json`
- `%AppData%\WinPomTimer\state.json`
- `%AppData%\WinPomTimer\sessions.jsonl`

## 8. よくある質問

- Q. `X` で閉じると終了する?
  - A. 終了せずトレイへ格納される（完全終了はトレイメニューから）
- Q. 何も起動しないように見える
  - A. 既にトレイ常駐している場合がある。タスクトレイを確認する
